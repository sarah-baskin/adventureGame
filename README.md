How does the game actually work?

A player is spawned in a room -- "Home"
They need to find the key that will let them escape

Each room has items in it, and adjacent rooms that you can explore.
Enemies may be hiding behind closed doors.

The items in "home" are a table, and on the table is a satchel and a letter

You can walk to the table, walk to the door on both sides, and pick up the satchel
